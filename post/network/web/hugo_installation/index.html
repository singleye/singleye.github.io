## 服务部署

第一步，使用 hugo 命令生成静态文件内容

$ cd <hugo project home>
$ hugo

执行完以上操作后会在项目目录中产生 public 目录，这个目录中包括了可以部署的静态文件。把 public 目录复制到 nginx 的公开目录中。

第二步，配置 nginx

server {
  listen       80;
  listen       [::]:80;
  server_name  yourhostname;

  location / {
    root /directory/of/public;
  }
}

至此，系统的搭建基本完成，赶紧访问吧 :-)
