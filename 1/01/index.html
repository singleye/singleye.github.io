## 流程框架

  1. 从 3D 点云中分割平面
  2. 把平面包含的点映射 project 到平面上
  3. 映射出的 3D 平面变换到 2D 坐标系平面上
  4. 从 2D 平面找出平行四边形
  5. 将平行四边形反向旋转恢复出 3D 空间中的平面四边形

## 从 2D 平面照出平行四边形的方法

# PCA检测主轴方向

可以找到平行四边形的两个主轴方向，但无法确定每个边的长度

问题：

  • 当电云分布不均匀时，导致PCA主轴方向向密度更高的方向略微偏移

# 边缘检测法

  1. 使用包络 (Graham Scan) 扫描出四边形的外围轮廓线
  2. 将靠近轮廓线的点距离小于某个阀值的所有点取出来
  3. 在取出的点中使用 RANSAC 估计4根直线
  4. 计算出 4 根直线的交点
