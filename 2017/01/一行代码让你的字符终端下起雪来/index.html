一行代码让你的字符终端下起雪来！

一月 15, 2017 发布在 Programming, Shell

在网上一段shell脚本，虽然只有一行但运行后可以在屏幕上实现下雪的效果，实在让人佩服，决定好好研究一下。

1

for((I=0;J=--I;))do clear;for((D=LINES;S=++J**3%COLUMNS,--D;))do printf %*s.\\n $S;done;sleep .1;done

不过原代码虽然简洁但不太容易一下子看明白工作原理，而且只能在命令行上运行无法保存成脚本文件，所以自己按照理解重新写了一遍：

#!/bin/sh

rows=$(tput lines)
cols=$(tput cols)

i=0
while true
do
    clear
    ((x=i,i++))
    for((r=0;r<rows;r++))
    do
        printf "%*s*\n" $((x**3%cols))
        ((x--))
    done
    sleep .1
done
